About

This project was made for the course Algorithms in my first year of my study, this involved setting up a dungeon with random room generation and door generation then generating nodes within this dungeon's rooms, drawing a grid between these nodes. Finally implementing both Recursive and Breadth-First Search algorithms for pathfinding.

Project Info

  • Duration: 4 Weeks
  • Engine: GXP Engine (C#)
  • Role: Algorithm Programming
  • Team size: 1
