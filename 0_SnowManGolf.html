About

A group project in my first year of university, the assignment was make a puzzle game based around physics. The term before this I made my own 2D physics engine so that's what we used for this project.

Project Info

  • Duration: 2 Weeks
  • Technology: MGE (C#)
  • Role: Gameplay Programmer

Project Details

This game was made in a group project with me as the sole engineer, we used the 2D physics engine I had developed for the course physics programming. This was all done in the MGE engine an in house engine made by my teachers at the time, It's like a lightweight unity but without the visual editor also in C#.
