About

This project was made for a course I had to do at uni, 3D rendering during this course I worked on implementing various mechanics and shaders like light reflections, attentuation, rendering terrain with a height/splat map and textures.

Project Info

  • Duration: 4 Weeks
  • Technology: OpenGL (C++)
  • Role: Shader Programmer, Engine Programmer

Project Details

The engine used is an inhouse engine that the course provided, its just a basic framework that provided the rendering of 3D objects and the ability to use vertex shaders and fragment shaders on certain materials using openGL without any additional features. The main features I worked on were camera movement not using any libraries but just math and openGL, implementing light and other effects into the shaders

Conclusion

During this project I learned a lot about the general graphics pipeline, OpenGL, deepend my knowledge about C++, and learned a lot about shaderprogramming. I also learned how a 3D render engine might look like behind the scenes which I actualy found very intriguing,
