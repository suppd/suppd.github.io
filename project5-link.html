Conclusion

This project taught me a lot about working in large codebases with a lot of diffrent code authors and how to approach this, it also taught me a lot about about how project management works in a corporate environment thanks to the project manager at Thales. I learned a lot about how we should define tasks when planning sprints learned a lot about scope by making mistakes when planning but atleast I learned from my mistakes which I think was a good thing.
