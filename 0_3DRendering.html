About

This project was made for a school module: 3D rendering, during this course I worked on implementing various features like light reflections and attentuation, rendering terrain with a height/splat map and textures.

Project Info

  • Duration: 4 Weeks
  • Technology: OpenGL (C++)
  • Role: Shader Programmer, Engine Programmer

Project Details

The engine used is an inhouse engine that the course provided, its just a basic framework that provided the rendering of 3D objects and the ability to use vertex shaders and fragment shaders on certain materials using openGL without any additional features. The main features I worked on were camera movement not using any libraries but just math and openGL, implementing lighting including light attentuation and reflections.

Conclusion

During this project I learned a lot about the general computer graphics pipeline both in theory and practice, I learned how to use the OpenGL library and deepend my C++ knowledge Furthermore I learned about shaderprogramming to render light on materials and more.

This project also taught me alot about how (3D) engines work and what goes into the backend of engine development.
